# 知识图谱可视化系统使用指南

这是一个基于Flask和HTML5 Canvas的知识图谱可视化系统，可以探索和搜索人工智能相关的知识图谱。

## 如何使用

### 启动服务器

```bash
cd /home/agentic/lite-webviewer/kg-agent
python3 kg_api_server.py
```

服务器将在 http://localhost:5004 上启动。

### 访问界面

在浏览器中打开 http://localhost:5004 即可访问知识图谱可视化界面。

### 使用方法

1. **探索知识图谱**：
   - 在搜索框中输入关键词（如"人工智能,机器学习"）
   - 点击"查询"按钮
   - 系统将显示相关节点及其关系

2. **搜索节点**：
   - 在搜索框中输入搜索关键词
   - 点击"搜索节点"按钮
   - 系统将显示匹配的节点列表
   - 点击搜索结果可查看相关节点图谱

3. **查看节点详情**：
   - 在图谱中点击任意节点
   - 在右侧信息面板中点击"查看详情"按钮
   - 系统将显示该节点的完整信息

## API端点

- `GET /` - 知识图谱可视化页面
- `POST /api/kg/explore` - 探索知识图谱
  - 请求体：`{"keywords": ["关键词1", "关键词2"], "depth": 2}`
- `GET /api/kg/node/<node_id>` - 获取节点详细信息
- `POST /api/kg/search` - 搜索知识图谱
  - 请求体：`{"query": "搜索关键词"}`
- `GET /health` - 健康检查

## 注意事项

- 当前系统使用模拟数据，实际应用中需要连接真实知识图谱数据库
- 系统支持中文关键词搜索和节点名称
- 节点详情功能需要对中文节点ID进行URL编码
